# Lottie Animation Setup Guide

## Download Required Animations

Visit [LottieFiles.com](https://lottiefiles.com) and download these FREE animations:

### Required Animations:
1. **loading.json** - Search: "loading spinner" or "circular loading"
2. **success.json** - Search: "success checkmark" or "success animation"
3. **empty.json** - Search: "empty state" or "no data"
4. **map_marker.json** - Search: "location pin" or "map marker"
5. **park.json** - Search: "park" or "tree nature"
6. **booking.json** - Search: "calendar booking" or "schedule"
7. **report.json** - Search: "report document" or "file upload"
8. **sync.json** - Search: "sync cloud" or "refresh"
9. **profile.json** - Search: "user profile" or "avatar"
10. **celebration.json** - Search: "celebration" or "confetti"

## Installation Steps:

1. Download each animation as JSON file
2. Rename them according to the list above
3. Place all files in: `assets/lottie/`
4. Run: `flutter pub get`

## File Structure:
```
assets/
  lottie/
    loading.json
    success.json
    empty.json
    map_marker.json
    park.json
    booking.json
    report.json
    sync.json
    profile.json
    celebration.json
```
